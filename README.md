# java-net-shell
通过java，实现一个用shell操作的网络上传、下载工具



## 实验要求

完成一个仿shell窗口，实现以下功能

1. **dir命令：**

   服务器返回指定目录下的文件列表

   输入格式如下：

   dir [path]

   

2. **get命令：**

   下载对应路径上的文件

   输入格式如下：

   get [path]
   

3. **put命令：**

   将本地路径上的文件，传输到服务器上

   输入格式如下：

   put [localPath] [remotePath]
   

4. **quie命令：**

   结束程序
